## Design Decisions
- **Skill-only implementation**: the entire define-truth flow is agent-driven prose. No TypeScript CLI command is needed — creating a markdown file in `taproot/global-truths/` is a trivial file-write the agent performs directly.
- **Pre-populated context path**: the skill accepts an optional `candidate` argument so that `/tr-discover-truths` and `/tr-ineed` can route promoted candidates here with term, scope, and evidence pre-filled. When no argument is provided, the skill asks interactively.
- **Scope determination is interactive**: the skill asks the developer to confirm scope rather than inferring it, to avoid silently applying the wrong scope to an important truth.
- **Both conventions supported equally**: the skill offers suffix and sub-folder as equal options and creates the file accordingly — no preference encoded.
- **Scope ambiguity flagged inline**: when a developer writes a truth without a clear scope signal, the skill surfaces the warning as part of the completion summary rather than as a blocker.
